AyoLomba!

June 12 2021 - Django HTML SQL

My project for Gemastik XIII Competition, my team created a web platform using Django to show and filter competition based on user's preference.

Project Brief

AyoLomba! is a web platform to accomodate event organizers and students across Indonesia to post and search competitions, scholarships, and webinars with better experience. Members can find the event they want based on their preferences, AyoLomba! also helps out organizers to reach their audience, using sleek UI and many features to help out both parties.
